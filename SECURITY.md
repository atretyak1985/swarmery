# Security Policy

## Reporting a Vulnerability

Please use [GitHub's private vulnerability reporting](https://github.com/atretyak1985/swarmery/security/advisories/new) to report security issues. Do not open a public issue.

You can expect an acknowledgement within 48 hours and a resolution or mitigation plan within 14 days.

## Supported Versions

Only the latest release on `main` is actively maintained.
